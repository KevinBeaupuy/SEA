Sarko Escape Adventure (SEA) est un escape game où on incarne un inspecteur qui enquête sur des affaires judicières de Nicolas Sarkozy.
Auteurs : Kévin Beaupuy et Félix Bal.

##
INSTALLATION / COMMENT JOUER

Il existe deux possibilités pour jouer à ce jeu : soit en téléchargeant les données du GitHub sur le lien https://github.com/KevinBeaupuy/SEA, puis en installant un logiciel comme Mamp. La seconde possiblité est de jouer sur le lien https://sarkos-escape-adventure.herokuapp.com/index.html, ce qui ne nécessite aucune installation. 


# SOLUTIONS

Lors du jeu un tableau résume ce qu'il faut faire. Pour certains objets, un indice supplémentaire est disponible en cliquant sur le bouton "indice supplémentaire". On ne peut pas cliquer sur ce bouton si aucun indice supplémentaire n'est disponible.
Il y a les solutions des objets à chercher dans l'onglet "solution" (cela ouvre un nouvel onglet).


# ACCES AUX BASES DE DONNEES EN LIGNE

Les bases de données utilisées sont stockées en ligne, on peut y accéder via ce lien : https://phpmyadmin.alwaysdata.com/phpmyadmin/index.php?route=/sql&db=kevineuh_sea&table=affaires&pos=0 .
Rentrer :
- Serveur : mysql-kevineuh.alwaysdata.net
- Utilisateur : kevineuh
- Mot de passe : root
